Demo Limitations

The demo is fully functional with two restrictions.

Demo Account Only

The demo EA can only be attached to charts on a demo trading account. It will not run on live/real accounts. Switch to a demo account in your MT5 terminal to use it.

10-Minute Sessions

Each session lasts 10 minutes. After 10 minutes the EA will stop automatically. Simply reattach it to the chart to start a new session — no limit on how many times you can restart.

How to Install

1

Download the .ex5 file

Click the download button above to get PineChart.ex5.

2

Copy to your MT5 Experts folder

In MetaTrader 5, go to File > Open Data Folder, then navigate to MQL5/Experts/ and paste the file there.

3

Attach to a chart on a demo account

Make sure you're on a demo account, then drag PineChart onto any chart. Enter your Pine Script filename in the InpIndicators input and click OK.
