## Operators

### Arithmetic Operators

| Operator | Status | Description|
|----------|--------|---------------|
| `+`      | ✅    | Addition      |
| `-`      | ✅    | Subtraction   |
| `*`      | ✅    | Multiplication|
| `/`      | ✅    |  Division     |
| `%`      | ✅    |  Modulo       |

### Comparison Operators

| Operator | Status | Description      |
|----------|--------|------------------|
| `==`     | ✅    | Equal             |
| `!=`     | ✅    |  Not equal        |
| `>`      | ✅    |  Greater than     |
| `<`      | ✅    |  Less than        |
| `>=`     | ✅    |  Greater or equal |
| `<=`     | ✅    |  Less or equal    |

### Logical Operators

| Operator | Status | Description  |
|----------|--------|--------------|
| `and`    | ✅    |  Logical AND |
| `or`     | ✅    |  Logical OR  |
| `not`    | ✅    |  Logical NOT |

### Assignment Operators

| Operator | Status | Description|
|----------|--------|---------------------|
| `=`      | ✅    |  Assignment         |
| `:=`     | ✅    | Reassignment        |
| `+=`     | ✅    | Add and assign      |
| `-=`     | ✅    | Subtract and assign |
| `*=`     | ✅    | Multiply and assign |
| `/=`     | ✅    | Divide and assign   |

### Series Operators

| Operator | Status | Description |
|----------|--------|----------|
| `[]` (index) | ✅ | Series access |
